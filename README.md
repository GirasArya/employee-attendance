
# Employee Attendance System

A simple Web Based employee attendance management system designed to track and manage employee attendance. It allows administrators to handle employee data, record attendance, and user management. 


## Tech Stack

**Client:** React, Axios, TailwindCSS, ChakraUI

**Server:** Node, Express, JWT

**Server:** MySQL


## Run Locally

Clone the project

```bash
  git clone https://github.com/GirasArya/employee-attendance
```

### Front end
Go to the project directory

```bash
  cd employee-attendance/Frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

### Backend
Go to the project directory

```bash
  cd employee-attendance/Backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```

## Environment Variables

To run this project, you will need to add the environment variables to your .env file (see .env.example for reference)
